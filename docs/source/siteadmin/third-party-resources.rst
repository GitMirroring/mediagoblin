Plugins
=======

.. csv-table::
   :header: Name,What,Options,Author

   `gmg_localfiles <https://github.com/Velmont/gmg_localfiles>`_,"Import files from your local filesystem without duplicating them.",,odinho/Velmont
   `mediagoblin_svg <https://github.com/commonsmachinery/mediagoblin_svg>`_,SVG image support,Display previews and thumbnails as SVG,,
   `mediagoblin-hidden_original <https://gitorious.org/mediagoblin-stock/mediagoblin-hidden_original>`_,Hidden original,"Give original upload a secret url, only publish downsized version",Unhammer 
   `mediagoblin-recaptcha <https://github.com/ayleph/mediagoblin-recaptcha>`_,`Google reCAPTCHA <https://www.google.com/recaptcha>`_ support,Enable reCAPTCHA on registration form over http/https,Ayleph
   `mediagoblin-piwik <https://github.com/ayleph/mediagoblin-piwik>`_,`Piwik <https://piwik.org>`_ support,,Ayleph
   `mediagoblin-basicsearch <https://github.com/ayleph/mediagoblin-basicsearch>`_,Search for media,,Ayleph
   `mediagoblin-embedcode <https://github.com/mayel/mediagoblin-embedcode>`_,Display embed code for video or audio player on media page,,mayel based on proof of concept by Ayleph


Plugins (work in progress)
==========================

You're working on a plugin but it's not ready? Please let us know.

.. csv-table::
   :header: Name,Planned features,Status,Author

   `Raw image support <https://github.com/Velmont/mediagoblin/tree/raw_image_mediatype>`_, "Support NEF [done], support CR2, investigate support for other formats (if they include full-quality JPEG inside the file)",,odinho/Velmont
   `Tag Cloud <https://gitorious.org/~spaetz/mediagoblin/spaetz-mediagoblin/trees/WIP/tag_cloud_plugin>`_,Add a tag cloud feature for your templates,See ``tag_cloud_plugin`` branch. It works but still needs core modifications.,Spaetz
   `HTML5-Multiupload <https://github.com/VenKamikaze/mediagoblin-html5-multi-upload>`_,Adds a the html5 multifile input to the submit form,"Working on 0.4.1, but needs proper review. Test and report bugs, it might not work, but it's absolutly harmless to your MG instance",kamikaze-m
   `MG-GIMP <http://chiselapp.com/user/saulgoode/repository/mg-gimp>`_,"Allows uploading of XCF images and GIMP resources (palettes, patterns, curves, gradients, brushes, dynamics, and tool presets). Thumbnails and previews are generated by an instance of GIMP running headlessly as a daemon",Nearing completion. Still needs to be packaged for easier deployment.,saulgoode
   `mg-rdfa <https://github.com/commonsmachinery/mg-rdfa>`_,Embedding image metadata as RDFa,"Experimental,only SVG images are supported at the moment",artfwo
   `uploadurl <https://github.com/sebastiansam55/uploadurl>`_,Upload Images via URL instead of file,Works when using browser that is logged in. Needs to use oauth or similar,sebastiansam55
